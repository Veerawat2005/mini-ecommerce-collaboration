ชื่อจิตรกร
แรงบันดาลใจหรือที่มาของรูปภาพ
มีลูกเล่น Css เล็กน้อยอย่างการปรับขนาดไซด์ภาพให้เท่ากัน
การนำเมาส์ไปชี้ที่รูปแล้วมีการแสดงภาำให้ลอยขึ้นมาเล็กน้อย
